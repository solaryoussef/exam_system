# Tic Tac Toe React

This project contains a simple Tic Tac Toe game built with React and Vite. It follows the official React tutorial and is intended as a learning resource.

## Available Scripts

- `npm run dev` – start a local development server
- `npm test` – run unit tests
- `npm run build` – create a production build
- `npm run preview` – preview the production build locally
